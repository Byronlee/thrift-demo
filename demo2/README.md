## Thrift Example

Thrift Example by Ruby client and Ruby server

## Usage

### use custom thrift client
```shell
cd rb
ruby server.rb
ruby client.rb
```
### use thrift client by twitter
```shell
gem install thrift_client
cd rb
ruby server.rb
ruby twitter_thrift_client.rb
```
